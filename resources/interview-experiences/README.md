## 面经结构

每个面经包含以下部分：

1. **背景**：面试者背景、目标岗位、公司信息
2. **面试流程**：轮次、形式、时长
3. **题目记录**：每轮面试的具体题目
4. **回答思路**：面试者的回答策略
5. **面试官反馈**：已知或推测的反馈
6. **结果**：通过/失败/待定
7. **复盘**：面试者的自我反思
8. **建议**：给后来者的建议

## 使用建议

### 面试前

1. **阅读目标岗位面经**：了解考察重点
2. **模拟练习**：用面经题目自我测试
3. **准备话术**：结合 [interview-patterns](../../interview-patterns/) 准备回答

### 面试后

1. **记录题目**：立即记录面试题目
2. **对比面经**：看看别人怎么回答
3. **复盘改进**：找出不足，针对性提升
