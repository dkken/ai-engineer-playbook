## 使用指南

### 求职者

1. **选项目前**：查看 [tools-comparison/](tools-comparison/) 了解技术选型
2. **做项目时**：参考 [awesome-ai-engineering-cn.md](awesome-ai-engineering-cn.md) 获取学习资源
3. **准备面试**：阅读 [case-studies/](case-studies/) 和 [interview-experiences/](interview-experiences/)

### 面试官

1. **设计面试题**：参考 [case-studies/](case-studies/) 获取真实场景
2. **评估候选人**：使用 [tools-comparison/](tools-comparison/) 考察技术选型能力
3. **了解竞品**：阅读 [interview-experiences/](interview-experiences/) 了解市场面试标准
