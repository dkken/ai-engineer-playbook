## 数据来源

海外数据基于 [ai-engineering-field-guide](https://github.com/alexeygrigorev/ai-engineering-field-guide) 对 **2,445份JD** 的量化分析（2026年1-3月，builtin.com）。

国内数据基于公开报告和社群征集样本（持续积累中）。

---

## 岗位类型分布

| 类型 | 占比 | 说明 |
|------|------|------|
| AI-First（直接构建AI系统） | 69.4% | 核心工作是RAG、Agent、LLM应用 |
| AI-Support（平台/基础设施） | 28.5% | 为AI团队提供基础设施支持 |
| ML-First（传统机器学习） | 2.1% | rebranded的ML岗位 |

**关键洞察**：近70%的AI工程师岗位直接参与AI系统构建，而非传统ML工作。

## 技能组合分析

### 最常见技能组合

1. **Python + RAG + AWS**（23.4%）— 构建云端RAG系统
2. **Python + Agents + Docker**（18.2%）— 构建容器化Agent应用
3. **Python + LLMs + CI/CD**（15.7%）— 工程化LLM应用
4. **Python + Evaluation + Monitoring**（12.3%）— AI质量保障

### 技能层级分布

```
基础层（几乎所有岗位）：
├── Python
├── Git
└── 基础软件工程

核心层（70%+岗位）：
├── RAG
├── Prompt Engineering
├── Docker
└── AWS/云基础

进阶层（30-50%岗位）：
├── Agents
├── LangChain/LangGraph
├── Vector DBs
├── Kubernetes
└── CI/CD

专家层（10-20%岗位）：
├── Evaluation框架
├── Fine-tuning
├── 多Agent系统
├── 模型推理优化
└── AI安全/对齐
```

## 技能需求趋势（2024-2026）

### 上升最快的技能

| 技能 | 2024年 | 2025年 | 2026年 | 增长率 |
|------|--------|--------|--------|--------|
| RAG | 15% | 28% | 35.9% | +139% |
| Agents | 5% | 10% | 14.4% | +188% |
| Evaluation | 4% | 8% | 11.1% | +178% |
| Vector DBs | 5% | 8% | 10.8% | +116% |

### 下降的技能

| 技能 | 2024年 | 2025年 | 2026年 | 变化 |
|------|--------|--------|--------|------|
| Fine-tuning | 15% | 12% | 9.2% | -39% |
| 传统ML（sklearn等） | 25% | 18% | 12% | -52% |
| 模型训练 | 20% | 12% | 8% | -60% |

**趋势解读**：
- 企业从"训练模型"转向"使用模型"
- RAG和Agent成为核心技能
- 评估能力从"加分项"变为"必备项"
- 传统ML技能需求下降，但不等于不需要

## 不同岗位类型的技能差异

### AI-First岗位（69.4%）

**核心技能**：
- RAG（45.2%）
- Agents（20.8%）
- Prompt Engineering（35.4%）
- Evaluation（15.9%）

**特点**：直接构建AI应用，需要强工程能力+AI产品思维

### AI-Support岗位（28.5%）

**核心技能**：
- Kubernetes（42.1%）
- AWS/Azure/GCP（48.7%）
- Monitoring（28.4%）
- Data Pipelines（31.2%）

**特点**：构建AI基础设施，需要强运维能力+系统架构

### ML-First岗位（2.1%）

**核心技能**：
- PyTorch/TensorFlow（85%）
- Fine-tuning（45%）
- Model Training（60%）
- Math/Statistics（70%）

**特点**：传统ML工作，需要强数学基础+模型训练经验

## 给求职者的技能建议

### 必备技能（必须掌握）

1. **Python** — 82.5%岗位提及，绝对基础
2. **RAG** — 35.9%岗位提及，最热门技能
3. **Docker** — 32.6%岗位提及，工程化基础
4. **Prompt Engineering** — 29.1%岗位提及，AI时代新技能

### 推荐技能（强烈建议）

5. **Agent开发** — 14.4%但增长最快（+188%）
6. **Evaluation** — 11.1%但差异化明显
7. **向量数据库** — 10.8%且RAG必备
8. **CI/CD** — 29.2%，工程化必备

### 加分技能（提升竞争力）

9. **Kubernetes** — 28.3%，高级工程化
10. **Fine-tuning** — 9.2%，特定场景需要
11. **多Agent系统** — 新兴，高级场景
12. **AI安全/评估** — 差异化，专家方向

### 国内特有建议

13. **国产大模型API** — 至少熟悉1-2个（通义/文心/智谱）
14. **国内云平台** — 阿里云或腾讯云基础
15. **合规意识** — 了解数据安全法和算法备案
