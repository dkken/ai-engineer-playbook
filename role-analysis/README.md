# 角色分析

> 角色侦察模块 —— 知己知彼，才能在面试和招聘中做出正确判断

---

## 定位

Role Analysis 回答两个核心问题：
- **对求职者**：AI 工程师到底做什么？和 ML 工程师有什么区别？JD 上的要求哪些是真的必须，哪些是"愿望清单"？
- **对面试官/HR**：我到底要招什么样的人？技能要求该怎么写？市场上有多少合格候选人？
