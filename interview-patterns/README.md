# 面试策略

> 面试执行模块 —— 从流程拆解到临场话术的完整战术

---

## 定位

Interview Patterns 不是教你"背答案"，而是帮你建立**面试思维框架**。你需要的是知道面试官在考察什么、怎么组织你的回答、如何把项目经验转化为面试加分项，而不是记住一套标准答案。

## 与实战项目的配合

本模块的最佳使用方式是：**边做项目，边准备面试**。

```
practical-projects/  完成一个项目
        ↓
        ├── 打开 02-project-deep-dive.md   →  为这个项目准备深挖话术
        ├── 打开 03-ai-system-design.md    →  思考这个项目的系统设计怎么讲
        └── 打开 05-behavioral-interview.md →  提炼项目中的行为面试素材
```
