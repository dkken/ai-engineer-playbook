# 认证路径与学习路线

> **使用说明**：根据 [self-assessment.md](self-assessment.md) 的得分和 [gap-analysis.md](gap-analysis.md) 的差距分析，获取个性化学习路线。
>
> 本路径将实战项目、面试准备、技能提升有机结合，提供"市场洞察 → 能力构建 → 面试表达"的完整闭环。

---

## 一、路径总览

```
阶段1: 基础构建（2-4周）
├── 项目: P1-01 企业级RAG知识库（覆盖 LLM 应用 + RAG）
├── 项目: P1-02 AI Agent任务调度器（覆盖 Agent 基础）
├── 面试: 准备项目深挖话术
└── 评估: 完成自测，基础层得分 > 60%

阶段2: 核心进阶（3-5周）
├── 项目: P1-03 LLM评估套件（覆盖 Evaluation）
├── 项目: P1-04 Prompt版本管理（覆盖 Prompt Engineering）
├── 项目: P2-01 多Agent协作系统（覆盖 Agent 进阶）
├── 面试: 准备系统设计题
└── 评估: 核心层得分 > 70%，完成2个Tier-2项目

阶段3: 差异化打造（4-6周）
├── 项目: P2-02 高级混合检索（覆盖 RAG 进阶）
├── 项目: P2-03 AI代码审查机器人（覆盖 工程化 + MLOps）
├── 项目: P2-04 RAG安全评估（覆盖 AI 安全）
├── 面试: 准备行为面试
└── 评估: 进阶层得分 > 60%，建立差异化优势

阶段4: 高级冲刺（6-8周）
├── 项目: P3-01 知识图谱RAG（覆盖 高级RAG）
├── 项目: P3-02 AI数据管道（覆盖 数据工程）
├── 项目: P3-03 生产级Agent平台（覆盖 系统架构 + MLOps）
├── 面试: 模拟面试
└── 评估: 专家层得分 > 50%，冲刺架构师岗位
```

## 二、按目标岗位推荐路径

### 路径A：AI应用工程师

**目标岗位**：AI应用开发、RAG工程师、Agent工程师
**核心技能**：RAG、Agent、Prompt Engineering、LLM应用开发
**薪资范围**：24-42万（国内）/ $120k-$200k（海外）
**市场需求**：国内大模型应用开发提及率 ~45%，最热门方向

| 周次 | 学习内容 | 实战项目 | 面试准备 | 覆盖自测维度 |
|------|----------|----------|----------|-------------|
| 1-2 | RAG原理与实现 | P1-01 | 项目深挖准备 | LLM 应用 + RAG |
| 3-4 | Agent架构与开发 | P1-02 | 系统设计基础 | Agent 开发 |
| 5-6 | LLM评估与优化 | P1-03 | 评估指标讲解 | 模型评估与优化 |
| 7-8 | Prompt工程进阶 | P1-04 | A/B测试案例 | Prompt Engineering |
| 9-10 | 多Agent协作 | P2-01 | 协作机制讲解 | Agent 开发 |
| 11-12 | 高级检索优化 | P2-02 | 检索算法对比 | RAG 系统 |

**关键产出**：
- 4个可运行的开源项目
- 2个进阶项目经验
- 完整的项目深挖话术
- 覆盖核心层全部 4 个维度

- [ ] 完成 P1-01 部署和至少2个功能扩展

- [ ] 完成 P1-02 和 P1-03

## 七、常见问题

### Q1：时间不够，如何取舍？

**建议**：
- 优先完成 Tier-1 的 P1-01 和 P1-02（市场最需求，覆盖 LLM + RAG + Agent）
- 选择1个 Tier-2 项目建立差异化（推荐 P2-04 安全评估 或 P2-03 MLOps）
- 如果时间极紧，只做 P1-01 + P1-02 + P2-04（安全是差异化，国内合规需求上升）
- 国内求职者务必补充国产大模型 API 调用（至少1个）

### Q2：没有项目经验，如何开始？

**建议**：
- 从 P1-01 开始，按照任务书逐步完成
- 每个项目至少完成"部署 + 一个功能扩展 + 基础监控"
- 记录遇到的问题和解决方案，这是面试素材
- 使用自测问卷每2周评估一次，跟踪进步

### Q3：如何平衡广度和深度？

**建议**：
- 广度：了解所有 Tier-1 项目的基本原理（12 个维度都了解）
- 深度：选择2-3个项目深入实践，能回答细节问题（目标维度达到 3-4 分）
- 面试时：广度展示知识面，深度展示实践能力
- 参考 [gap-analysis.md](gap-analysis.md) 中的"常见差距模式"，避免典型陷阱

### Q4：如何验证学习效果？

**建议**：
- 每完成一个项目，用自测问卷对应维度重新评分
- 使用 [radar-chart.html](radar-chart.html) 可视化能力变化
- 找朋友进行模拟面试，重点练习项目深挖
- 参与开源社区，获取代码审查反馈
- 里程碑验收：基础层 > 60%，核心层 > 70%，进阶层 > 60%

### Q5：国内求职和海外求职有何不同？

**建议**：
- **国内特有要求**：国产大模型（通义/文心/智谱至少熟悉1个）、国内云平台（阿里云/腾讯云）、算法备案和数据安全法
- **海外侧重**：分布式系统经验、开源贡献、英语沟通能力
- **薪资差异**：海外整体更高，但国内大厂（字节/阿里）高级岗位薪资接近海外
- **技能溢价**：国内 AI 解决方案溢价 +30%，海外 Distributed Systems 溢价 +$34K
